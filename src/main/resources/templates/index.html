시스템 개요

이 애플리케이션은 Model Context Protocol(MCP)을 활용하여 문서 처리 및 데이터 시각화 통합 시스템을 제공합니다. 다양한 문서 형식을 처리하고 데이터를 시각화하여 유용한 인사이트를 제공합니다.

주요 기능

  • 다양한 문서 형식 파싱 (PDF, Excel, Word)
  • 구조화된 데이터 추출 및 분석
  • 자연어 기반 시각화 요청 처리
  • 인터랙티브 데이터 탐색
  • 파일 시스템 작업 (읽기, 쓰기, 목록 조회 등)
